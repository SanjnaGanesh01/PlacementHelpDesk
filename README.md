PlacementHelpDesk

This project is a **Java console program** that helps students explore placement roles and prepare effectively.

 Features
- Interactive menu to view placement roles
- Displays required skills and preparation tips for each role
- Easy to expand with new roles in the future

 Technologies Used
- Java
- Console-based interaction

 How to Run
1. Clone or download the repository  
2. Open the project in any Java IDE (Eclipse, IntelliJ, etc.)  
3. Run `PlacementGuide.java`  
4. Follow the on-screen menu to explore roles

 Use Case
- Student placement guidance
- Resume & skill preparation
- Learning object-oriented programming concepts
